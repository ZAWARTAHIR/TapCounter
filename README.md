# 👆 Tap Counter Pro

A modern, highly responsive, and beautifully designed Tap Counter application built with **React** and **Vite**. Features a sleek dark-mode glassmorphism aesthetic with engaging micro-animations.

## ✨ Features

- **Beautiful UI**: Modern dark mode with glassmorphism effects and animated background gradients.
- **Micro-Animations**: Bouncy text animations, satisfying button presses, and pulsing hints.
- **Fully Responsive**: Perfectly adapts to all screen sizes, from ultra-wide desktops to tiny mobile screens.
- **Core Functionality**: 
  - Tap anywhere to increment the counter
  - Dedicated `Decrement` button
  - `Reset` button

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

You need [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository**:
   ```bash
   git clone <your-repo-url>
   cd tap-counter
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser**:
   Navigate to the local server URL provided in your terminal (usually `http://localhost:5173/`).
